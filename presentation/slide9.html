Conclusion

Vers un marché immobilier plus transparent en Mauritanie

Dataset Inédit

Premier dataset immobilier structuré de Nouakchott : 1 153 annonces enrichies géographiquement.

R² = 0.84

XGBoost atteint 84% de variance expliquée avec une MAE de 780K MRU sur les prédictions.

Insights Marché

Quartier, surface et type de bien : les 3 déterminants majeurs des prix à Nouakchott.

Pipeline Complet

De la collecte au modèle déployé : un pipeline end-to-end reproductible.

Limites

  • Dataset limité à une seule source (Voursa.com)
  • Valeurs manquantes sur surface et pièces (~15%)
  • Pas de données temporelles (évolution des prix)
  • Géocodage imprécis pour certains quartiers périphériques

Améliorations Futures

  • Multi-sources : ajouter d'autres plateformes immobilières
  • Application web Streamlit pour les agents immobiliers
  • Données satellites (images) pour enrichir les features
  • Suivi temporel des prix (time series forecasting)
